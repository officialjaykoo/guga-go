# 구가바둑 룰
- 바둑판 크기: 19 x 13
- 1~4수(패스 포함)는 중립돌(초록) 배치: 흑 중립 → 백 중립 → 흑 중립 → 백 중립
- 패스도 초반 4수에 포함되며, 이 경우 초록돌이 4개 미만일 수 있다
- 중립돌은 최대 4개이며 위치는 흑/백이 자유롭게 결정
- 초반 4수는 중립돌 배치 단계로 단수/따냄/자살 체크 없음
- 5수부터 흑/백이 정상적으로 교대로 둔다
- 중립돌은 제3의 돌이며 흑/백과 연결되지 않는다
- 중립돌은 캡쳐/사석 처리가 불가능하다
- 중립돌은 집을 만들지 못하며 자리만 차지한다
- 중립돌은 벽처럼 취급되며, 중립돌에 닿아도 흑/백 단색이면 집이 될 수 있다
- 사석 자동 제안은 참고용이다
- 사석 확정/분쟁 처리는 별도 규정 없이 합의로만 진행한다
- 계가 거절 시 대국을 계속하며, 수락 시 즉시 종료된다
- 초읽기: 30초 3회
- 무르기 요청: 게임당 각 플레이어 3회까지
- 계가 요청: 100수 이상, 자기 턴에서만 가능
- 계가 요청은 15초 내 응답, 미응답 시 수락으로 간주
- 덤: 0

## 중립돌(초록) 핵심 요약
- 중립돌은 벽처럼 취급되는 제3의 점유물이다
- 빈칸을 점유하며 흑/백과 연결되지 않는다
- 중립돌에 닿아도 흑/백 단색이면 집이 될 수 있다
- 중립돌은 캡쳐/사석 처리가 불가능하다
- 중립돌끼리는 자살/캡쳐 판정이 없다
